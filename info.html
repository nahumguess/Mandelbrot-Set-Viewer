Mandelbrot Set Info

The Mandelbrot Set is a set of complex numbers that has a highly complex and fractal boundary when plotted in the complex plane. It was discovered by Benoit Mandelbrot in 1978 and has been a subject of study and fascination ever since.

The Mandelbrot Set is generated by iterating a simple mathematical equation over and over again for each point in the complex plane. If the iteration reaches infinity for a given point, that point is considered to be outside of the Mandelbrot Set. If the iteration remains bounded for a given point, that point is considered to be inside the Mandelbrot Set.

The equation that produces the Mandelbrot Set is Zn+1 = Zn2 + C.

What makes the Mandelbrot Set so interesting is the incredible detail and complexity of its boundary. The boundary contains intricate and beautiful fractal patterns that exhibit self-similarity at all scales. In fact, the boundary is infinitely complex and contains an infinite amount of detail!
